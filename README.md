# Relocation & Finance Calculator 💶📦

## Overview
This is a command-line application built in Python to calculate the financial impact of relocating. It handles the complex overlapping costs associated with managing old and new rental contracts simultaneously, along with one-off moving expenses.

## Features
*   **Dual-Rent Calculation:** Computes the exact financial burden of overlapping lease periods.
*   **Expense Aggregation:** Factors in moving service lump sums for a total relocation cost overview.
*   **Command-Line Interface:** Easy-to-use terminal prompts for quick data entry.

## Technologies Used
*   Python 3.x
